#Python Network Port Scanner & Baseline Monitor

A multi-threaded Python-based port scanner and network monitoring tool that detects open ports, identifies common service vulnerabilities, and supports baseline comparison for anomaly detection.

Some Features included in this project are:
-Multi-threaded TCP port scanning
-Detects commonly exposed services (HTTP, SSH, FTP, RDP, etc.)
-Built-in vulnerability reference for well-known ports
-Generates structured scan reports
-CLI-based custom port range scanning
-Baseline mode for detecting newly opened or closed ports
-JSON-based baseline storage
-Fast concurrent scanning using ThreadPoolExecutor

Inbuilt libraries in python3 like argparse, datetime, os, and socket were used for this project.
